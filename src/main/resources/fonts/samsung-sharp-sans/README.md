# Samsung Sharp Sans deployment asset

Samsung Sharp Sans is proprietary and is intentionally **not** distributed with Potaty.

Potaty's CSS prefers a locally installed `Samsung Sharp Sans` or `SamsungOne` and then falls back to the bundled OFL-licensed Pretendard variable font and open system fonts. If you hold a valid webfont licence, provide your licensed WOFF2 files during deployment and add a private deployment stylesheet that declares the same `Samsung Sharp Sans` family.

Do not commit the font binaries to this Apache-2.0 repository unless your licence explicitly permits redistribution.
